# 习惯打卡小程序 - 产品需求文档

## 一、产品概述
一款帮助用户养成良好习惯的微信小程序，通过简单的打卡 + 激励机制，让坚持变得有趣且有成就感。适用于学习、运动、阅读、练琴等各种需要打卡的场景。

## 二、核心功能

### 1. 打卡功能（首页核心）
- 大按钮：今日打卡
- 显示今日是否已打卡
- 打卡类型选择：练琴、运动、跳舞、其他（支持自定义输入）
- 打卡备注：记录今天做了什么

### 2. 打卡统计卡片
- 连续打卡天数
- 最长打卡记录
- 累计打卡天数
- 本周时长统计（小时）
- 支持按类型筛选统计（练琴、运动、跳舞等）

### 3. 打卡日历
- 月历视图
- 绿色渐变标记已打卡日期（带勾号）
- 半透明白色表示未打卡
- 今天未打卡：蓝色边框
- 今天已打卡：绿色渐变（和其他打卡日期一样）
- 支持月份切换

### 4. 激励机制（重点）
**连续3天打卡：**
- 显示笑脸表情 😊
- 文案："太棒了！连续3天打卡，继续加油！"

**连续7天打卡：**
- 显示小红花 🌺
- 文案："坚持一周啦！你真棒！"
- 可分享给老师/朋友

**连续15天打卡：**
- 显示勋章 🏅
- 文案："半个月的坚持，你已经超越了90%的人！"
- 显示励志语录
- 可分享成就

### 5. 打卡详情记录
- 打卡类型：4个预设（练琴、运动、跳舞、其他）+ 自定义输入（必填）
- 记录时长（分钟，纯数字输入）（必填）
- 上传照片或视频（可选，最多3个，单个文件最大10MB）
- 记录内容（可选，比如：练了哈农、599、某首曲子；或者读了某本书第几页）
- 心得体会（可选，记录今天的感受和收获）

### 6. 历史记录页面
- 显示所有打卡记录
- 按日期倒序排列
- 显示照片/视频缩略图
- 显示时长和内容
- 显示心得体会（如果有）
- 点击查看详情

## 三、设计风格

### 设计风格

### 配色方案
- 主色调：淡蓝色 (#87CEEB)
- 辅助色：向日葵黄 (#FFD700)
- 背景：雪山照片 + 淡蓝色渐变叠加
- 打卡日历：草绿色渐变 (#9ACD32 到 #7CB342)

### 视觉元素
- 雪山背景照片（真实照片，来自 Unsplash）
- 半透明白色卡片（毛玻璃效果）
- 圆润的卡片设计（24rpx 圆角）
- 北欧冷淡风格
- 大字体设计（标题 48rpx，数字 56-64rpx）

### 字体
- 页面标题：48rpx，加粗
- 统计数字：56rpx，加粗
- 时长输入：64rpx，加粗，居中
- 正文：28-32rpx
- 小字：26rpx

## 四、技术实现

### 页面结构
```
pages/
├── index/          # 首页（打卡按钮 + 统计 + 日历）
├── checkin/        # 打卡详情页（上传照片/视频、填写时长和内容）
└── history/        # 历史记录页（查看所有打卡记录）
```

### 数据存储
- 使用 wx.setStorageSync 存储打卡记录（后期可升级到云开发）
- 数据结构：
```json
{
  "checkInRecords": [
    {
      "id": "1",
      "date": "2026-02-19",
      "type": "练琴",
      "duration": 30,
      "content": "哈农、599",
      "media": ["图片路径或视频路径"],
      "reflection": "今天练得很顺利，手指灵活度提升了"
    }
  ],
  "achievements": ["3days", "7days", "15days"]
}
```

### 激励触发逻辑
1. 每次打卡后计算连续天数
2. 达到里程碑时弹出祝贺弹窗
3. 记录已获得的成就
4. 支持分享到微信

## 六、用户流程

1. 用户打开小程序 → 看到首页（打卡按钮 + 日历 + 统计）
2. 点击"今日打卡"按钮
3. 进入打卡详情页：
   - 选择打卡类型（练琴/运动/跳舞/其他，或自定义输入）（必填）
   - 填写时长（分钟，纯数字）（必填）
   - 可选：上传照片或视频（最多3个）
   - 可选：填写内容（今天做了什么）
   - 可选：填写心得体会
4. 提交打卡
5. 系统记录打卡，更新日历和统计（包括本周时长）
6. 如果达到里程碑（3天/7天/15天），弹出祝贺
7. 用户可以查看历史记录
8. 日历上今天的日期显示为绿色（已打卡）或蓝色边框（未打卡）

## 七、成功指标

- 用户连续打卡率 > 30%
- 7天留存率 > 50%
- 分享率 > 20%
- 用户平均打卡时长 > 20分钟/天
